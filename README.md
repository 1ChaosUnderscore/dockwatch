A real-time Docker container health monitoring from your terminal

## Requirements
- Python 3.8+
- Docker running locally

## Installation

pip install git+https://github.com/1ChaosUnderscore/dockmira.git

## Features
- Live CPU% per container
- Memory usage with cached accuracy
- Network I/0- (upload/download in KB)

### Commands

- watch - watches all live containers
- list - lists all currect containers only once
